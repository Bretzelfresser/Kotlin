
HOW TO  COMPILE TO COMAND LINE:
     - navigate to the location of the PP_Wiegel_Wannke_Koppenhoefer project folder 
	 e.g. "cd C:\Users\[user]\Dokumente\GitHub\PP_Wiegel_Wannke_Koppenhoefer"
     - execute the gradle script using "gradlew jar"
     - navigate to the newly compiled jar-file in 
	 .\PP_Wiegel_Wannke_Koppenhoefer\build\libs\PP_Wiegel_Wannke_Koppenhoefer-2.0
     - execute the script with the directory of the graph data 
	 eg "java -jar PP_Wiegel_Wannke_Koppenhoefer-2.0.jar -graph /home/[user]/germany.fmi"

then open your browser and visit the address "localhost:8080".
Have fun!
